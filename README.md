# Algoritmo-Genetico-tsp
Implementação do Algoritmo Genético aplicado ao Problema do Caixeiro Viajante (TSP) com 20 cidades. Utiliza seleção por roleta, crossover cíclico (CX) e mutação por troca. Desenvolvido em Python com NumPy e Matplotlib. Inclui relatório no console, gráfico de convergência e visualização da melhor rota.
